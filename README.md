# iSuite Assistant Desktop

Desktop version of iSuite Assistant with license key authentication and local file operations.

## Features

- License key authentication (no Clerk needed)
- Local file operations via Composio
- AI chat with Claude
- Chat history synced to Convex
- Standalone installer (no dev servers needed)

## Build Instructions

### Windows
```bash
npm run build:frontend
npm run build:win
```

### Mac (via GitHub Actions)
Push to GitHub and the Mac build will run automatically.

Or build locally on Mac:
```bash
npm run build:frontend
npm run build:mac
```
